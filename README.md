# ネタバレーション

## サイト概要
### サイトテーマ
ネタバレOKなコミュニティサイト
​
### テーマを選んだ理由

映画やドラマ、ゲームを楽しむ中で、視聴後の興奮や考察を誰かと共有したいというとはよくあることですが、昨今SNSではネタバレの配慮をする人が増えています。私もネタバレ厳禁なTRPGやマーダーミステリーで遊ぶことが多く、自由に感想や考察を投稿できないことに不便さを感じていました。

特に「ネタバレを語りたい側」と「ネタバレを避けたい側」が同じ場所にいるSNSでは、どちらも遠慮が生まれ、率直なコミュニケーションが取りづらいという課題があります。

そこで、ネタバレを前提としたコミュニティがあれば、同じ作品を楽しんだユーザー同士が気兼ねなく語り合える場所になると考え、このテーマを選びました。
​
### ターゲットユーザ
- 感想を深掘りしたい人
- 考察・意見交換を楽しみたい人
- 作品の熱量を共有できる場所を探している人
​
### 主な利用シーン
- ネタバレを前提とした感想を気軽に発信したい時
- 他のユーザーの考察や感想を読んで新しい視点を得たい時
